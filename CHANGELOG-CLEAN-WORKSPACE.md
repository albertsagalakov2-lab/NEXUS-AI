# Clean Workspace redesign

## Изменено

- Убрана перегруженная правая панель из чата.
- Добавлена минималистичная desktop-панель шириной 248 px.
- Добавлены пункты: новый чат, поиск, AI-агент, проекты, медиа, изображения, видео, аудио, музыка и приложения.
- Сохранена история чатов с открытием, переименованием и удалением.
- Добавлено мобильное выдвижное меню.
- Переработана нижняя мобильная навигация.
- Пустой чат теперь содержит компактный маскот, единое поле ввода и четыре быстрых действия.
- Сообщения оформлены в спокойные компактные bubbles.
- Сохранена фирменная сине-фиолетовая палитра, но уменьшено количество неоновых эффектов.
- Сохранены маршруты, Supabase и существующие API.

## Проверено

- `npm run typecheck`
- `npm run build`

## Упрощённая навигация

- Удалены пункты «Проекты», «Медиа» и «Приложения» из бокового и мобильного меню.
- Сохранены отдельные разделы «Изображения» и «Видео».
- Мобильная нижняя навигация теперь состоит из четырёх элементов.
