# Как установить финальную версию NeiroPeiro AI

1. Распакуйте архив.
2. Скопируйте всё содержимое папки `NeiroPeiro-AI-final` в локальную папку репозитория GitHub.
3. Разрешите замену файлов.
4. В GitHub Desktop создайте коммит, например: `Финальный desktop и mobile landing`.
5. Нажмите `Push origin`.
6. Vercel автоматически создаст новый деплой.

## Маршруты

- `/` — публичная главная страница.
- `/sign-in` — вход.
- `/sign-up` — регистрация.
- `/chat` — чат.
- `/image` — генерация изображений.
- `/video` — генерация видео.

## Переменные окружения

При обновлении существующего Vercel-проекта повторно добавлять переменные не нужно. Проверьте, что уже настроены:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- переменная API-ключа, которую читает ваш проект
- URL API и название модели, если они используются

Не загружайте `.env` и секретные ключи в GitHub.

## Как откатить

### GitHub Desktop

1. Откройте `History`.
2. Нажмите правой кнопкой по коммиту с обновлением.
3. Выберите `Revert Changes in Commit`.
4. Нажмите `Push origin`.

### Vercel

Откройте `Deployments`, выберите старый рабочий деплой и выполните `Promote to Production`.
